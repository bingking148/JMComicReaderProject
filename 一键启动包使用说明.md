# 🎨 JM漫画阅读器 - 一键启动包使用指南

## 📦 快速开始

### Windows 用户（推荐）

1. **下载启动包**
   - 将 `JMComicReader_Portable` 文件夹解压到任意位置

2. **一键启动**
   - 双击 `启动.bat` 文件
   - 等待服务启动完成

3. **开始使用**
   - 在浏览器中访问 `http://localhost:5000`

### macOS 用户

1. **下载启动包**
   - 将 `JMComicReader_Portable` 文件夹解压到任意位置

2. **启动应用**
   - 方式一：双击 `JMComicReader.app` 应用
   - 方式二：在终端运行 `./启动.sh`

3. **首次运行**
   - 右键点击应用 → 选择"打开" → 点击"打开"（解决安全提示）

4. **开始使用**
   - 在浏览器中访问 `http://localhost:5000`

### Linux 用户

1. **下载启动包**
   - 将 `JMComicReader_Portable` 文件夹解压到任意位置

2. **添加执行权限**
   ```bash
   chmod +x 启动.sh
   chmod +x JMComicReader
   ```

3. **启动应用**
   ```bash
   ./启动.sh
   ```

4. **开始使用**
   - 在浏览器中访问 `http://localhost:5000`

## ⚙️ 功能说明

### 1. 搜索漫画

#### JM号搜索（精准搜索）
- 在首页输入框中输入 JM 号
- 点击"搜索"按钮
- 快速定位指定漫画

#### 关键词搜索（模糊搜索）
- 点击"关键词搜索"按钮
- 输入关键词
- 选择排序方式（最新/最热）
- 浏览搜索结果

### 2. 下载漫画

1. 在搜索结果中找到想要的漫画
2. 点击"详情"查看完整信息
3. 点击"下载"按钮开始下载
4. 查看下载进度条
5. 下载完成后可在"已下载"页面查看

### 3. 阅读漫画

1. 在"已下载"页面找到已下载的漫画
2. 点击"阅读"按钮
3. 使用阅读器功能：
   - **← / →** 上一页 / 下一页
   - **+ / -** 放大 / 缩小
   - **F** 全屏
   - **ESC** 退出全屏 / 返回
4. 多章节漫画可在左侧切换章节

### 4. 管理漫画

- **查看已下载**: 访问"已下载"页面
- **删除漫画**: 点击删除按钮移除不需要的漫画
- **查看信息**: 显示文件大小、下载时间等信息
- **清理缓存**: 在设置中清理临时缓存

## 🛠 故障排除

### Windows

#### 1. 端口被占用
```
错误: 端口 5000 已被占用
```

**解决方案：**
```cmd
netstat -ano | findstr :5000
taskkill /F /PID <进程ID>
```

或修改 `启动.bat` 中的端口设置：
```cmd
set "APP_PORT=5001"
```

#### 2. Python 未安装
```
错误: 未找到 Python 或版本不符合要求
```

**解决方案：**
1. 访问 https://www.python.org/downloads/
2. 下载并安装 Python 3.8 或更高版本
3. **重要**: 安装时勾选 "Add Python to PATH"
4. 重新运行 `启动.bat`

#### 3. 依赖安装失败
```
错误: 依赖安装失败
```

**解决方案：**
1. 手动运行 `install_deps.py` 安装依赖
2. 或手动安装：
```cmd
pip install flask flask-cors jmcomic requests pillow img2pdf aiofiles aiohttp
```

#### 4. 防火墙拦截
```
无法访问 localhost:5000
```

**解决方案：**
1. 打开 Windows 防火墙设置
2. 允许 Python 通过防火墙
3. 或临时关闭防火墙测试

### macOS

#### 1. 无法打开应用（安全提示）
```
"JMComicReader.app" 已损坏，无法打开
```

**解决方案：**
1. 右键点击应用 → 选择"打开"
2. 在弹出的安全提示中点击"打开"
3. 或在系统设置中允许：
   - 系统设置 → 隐私与安全性 → 允许从以下位置下载的应用

#### 2. Python 未安装
```
错误: Python not found
```

**解决方案：**
```bash
brew install python3
```

#### 3. 端口被占用
```bash
# 查找占用进程
lsof -i :5000

# 结束进程
kill -9 <进程ID>
```

### Linux

#### 1. 权限不足
```
bash: ./启动.sh: Permission denied
```

**解决方案：**
```bash
chmod +x 启动.sh
chmod +x JMComicReader
```

#### 2. Python 未安装

**Ubuntu/Debian:**
```bash
sudo apt-get update
sudo apt-get install python3 python3-pip
```

**CentOS/RHEL:**
```bash
sudo yum install python3 python3-pip
```

#### 3. 端口被占用
```bash
# 查找占用进程
sudo lsof -i :5000

# 结束进程
sudo kill -9 <进程ID>
```

## 🔧 高级配置

### 修改端口

编辑 `启动.bat` 或 `启动.sh`：

**Windows (启动.bat):**
```cmd
set "APP_PORT=5001"
```

**macOS/Linux (启动.sh):**
```bash
APP_PORT=5001
```

### 修改 JM 域名

编辑 `backend/jm_option.yml`：

```yaml
client:
  domain:
  - www.cdnhth.club
  - www.cdngwc.cc
  - # 添加更多域名...
```

### 修改下载路径

编辑 `启动.bat` 或 `启动.sh`，添加环境变量：

**Windows:**
```cmd
set "DOWNLOAD_DIR=自定义路径"
set "TEMP_CACHE_DIR=自定义缓存路径"
```

**macOS/Linux:**
```bash
export DOWNLOAD_DIR="自定义路径"
export TEMP_CACHE_DIR="自定义缓存路径"
```

## 💡 使用技巧

### 1. 批量下载
- 搜索关键词后，可以逐个下载感兴趣的漫画
- 已下载的漫画会自动保存，无需重复下载

### 2. 缓存管理
- 定期清理 `TempCache` 可以释放磁盘空间
- 已下载的漫画不会受影响

### 3. 章节切换
- 多章节漫画支持快速切换
- 自动保存阅读进度

### 4. 局域网共享
- 在同一局域网内，其他设备可以通过你的 IP 地址访问
- 适合在手机或平板上阅读

## 🔒 安全说明

1. **网络安全**
   - 仅在受信任的网络环境中使用
   - 不要在公共网络中开放端口访问

2. **数据安全**
   - 定期备份 `DownloadedComics` 目录

   - 应用不会上传任何数据到第三方服务器
   - 所有数据仅存储在本地

## 📞 技术支持

### 常见问题

**Q: 为什么下载很慢？**
A: 下载速度取决于网络连接和 JM 服务器状态，请耐心等待或尝试更换网络环境。

**Q: 可以离线阅读吗？**
A: 已下载的漫画可以离线阅读，但搜索功能需要网络连接。

**Q: 支持哪些图片格式？**
A: 支持 JPG、PNG 等常见图片格式，自动转换为 JPG 存储。

**Q: 如何备份数据？**
A: 备份 `DownloadedComics` 目录和 `backend/comics.db` 数据库文件即可。
